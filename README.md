## ปัญหา

ทีม influencer marketing ของเอเจนซี่ต้องเจอปัญหาเหล่านี้ทุกครั้งที่รับลูกค้าใหม่:

1. **หา KOL ที่เหมาะกับแบรนด์ด้วยมือ** — ไล่ดูโปรไฟล์ทีละคนใน TikTok/YouTube ใช้เวลาเป็นวัน
2. **ไม่รู้ว่า follower เป็นของจริงหรือซื้อมา** — ตัวเลข follower สูงไม่ได้แปลว่า engagement ดีจริง
3. **พลาดเลือกครีเอเตอร์ที่เนื้อหาเสี่ยง** — เคยโพสต์เรื่องการเมือง/พนัน/ไม่เหมาะสม กระทบภาพลักษณ์แบรนด์ทีหลัง
4. **ตัดสินใจงบประมาณโดยไม่มีตัวเลขอ้างอิง** — ไม่รู้ว่าจ้างคนนี้แล้วจะได้ยอดวิว/ROI ประมาณเท่าไหร่
5. **เจาะกลุ่มเป้าหมายเฉพาะทางไม่ได้** — เช่น อยากได้ influencer ที่เข้าถึงกลุ่มนักศึกษามหาวิทยาลัยใดมหาวิทยาลัยหนึ่งโดยเฉพาะ เครื่องมือทั่วไปทำไม่ได้ละเอียดขนาดนั้น

## โซลูชัน

| ปัญหา | ฟีเจอร์ที่แก้ |
|---|---|
| หาด้วยมือช้า | วางลิงก์เว็บไซต์ → AI วิเคราะห์แบรนด์ → ค้นหา KOL อัตโนมัติทั้ง TikTok และ YouTube |
| follower ปลอมไม่รู้ตัว | ตรวจ engagement rate เทียบ benchmark ตามระดับ follower + เทียบกับเพื่อนร่วมหมวดหมู่ |
| เนื้อหาเสี่ยงหลุดตา | AI (Claude) อ่านแคปชั่นล่าสุดของ KOL เช็คเนื้อหาการเมือง/พนัน/18+/hate speech |
| ตัดสินใจงบไม่มีข้อมูล | ทำนายยอดวิว + คำนวณ ROI จากสมมติฐาน CTR/conversion ที่ทีมมาร์เก็ตติ้งปรับเองได้ |
| เจาะกลุ่มเป้าหมายเฉพาะ | ช่อง "บริบทเพิ่มเติม" ให้พิมพ์กลุ่มเป้าหมายที่ต้องการเจาะจง (เช่น มหาวิทยาลัยใดมหาวิทยาลัยหนึ่ง) แล้ว AI จะให้ priority กับสิ่งนี้เป็นอันดับแรกในการค้นหา — เหนือกว่าคำจากเว็บไซต์เองด้วยซ้ำ |

## เทคโนโลยีที่ใช้

| ส่วน | เทคโนโลยี |
|---|---|
| Backend | Python + FastAPI |
| Frontend | HTML/CSS/JS ล้วน |
| AI วิเคราะห์แบรนด์/ความเสี่ยง | Claude API (Anthropic) |
| ข้อมูล TikTok | Apify (TikTok Scraper actor) |
| ข้อมูล YouTube | YouTube Data API v3 |
| ดึงข้อความเว็บไซต์ | trafilatura + BeautifulSoup |
| Deployment | Docker + docker-compose |


## ฟีเจอร์เด่น

- 🔍 **ค้นหาข้ามแพลตฟอร์ม** — เลือกได้ TikTok, YouTube, หรือทั้งคู่พร้อมกัน
- 🎯 **บริบทกำหนดกลุ่มเป้าหมายเอง** — พิมพ์ระบุกลุ่มเป้าหมายเฉพาะได้ (priority สูงสุด เหนือเนื้อหาเว็บไซต์)
- 🛡️ **ตรวจ 2 ชั้น** — heuristic คัดกรองราคาถูกก่อน แล้ว AI ยืนยันเฉพาะผู้เข้ารอบ (ประหยัด cost การเรียก AI)
- ⚠️ **Risk analysis ครบวงจร** — fake follower, engagement ผิดปกติ, เนื้อหาเสี่ยงต่อแบรนด์
- 📊 **ทำนายผลก่อนตัดสินใจ** — ยอดวิว, ROI โดยประมาณ, เทียบกับคู่แข่งในหมวดหมู่เดียวกัน
- ⚡ **สถานะ real-time** — เห็นทุกขั้นตอนขณะประมวลผล ไม่ใช่หน้าจอเงียบรอ
- 🔄 **Graceful fallback** — ไม่มี API key ก็ยังทำงานได้ (heuristic + ข้อมูลจำลอง) ไม่มีวันล่ม



## Roadmap

- [ ] ปรับ "บริบทเพิ่มเติม" ให้แปลงเป็นคำค้นที่ TikTok/YouTube search เข้าใจง่ายขึ้น (ตอนนี้เป็นประโยคเต็ม)
- [ ] ระบบติดตามผลหลังแคมเปญจริง (ยอดวิว/engagement จริงหลังจ้าง) + ให้ระบบเรียนรู้แม่นขึ้นเรื่อยๆ
- [ ] ระบบคำนวณต้นทุน-กำไรแบบเต็มรูปแบบ (ใส่ข้อมูล conversion rate จริงของลูกค้าแต่ละราย มาคำนวณว่าจ้าง KOL คนไหนคุ้มทุนที่สุด)
